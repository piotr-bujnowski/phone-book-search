# phonebook-search-algorithms
This is a little experiment with search and sort algortithms for phone book searching.  
**For searching 500 contacts in around 1 mln record phone-book was used/tested:** 
- LinearSearch
- JumpSearch
- BinarySearch
- HashTableSearch  

**For sorting:**
- QuickSort
- BubbleSort
